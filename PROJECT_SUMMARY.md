## 💡 Customization Guide

### Update Personal Information

1. Edit email: `src/pages/Contact.jsx`
2. Edit LinkedIn/GitHub: `src/components/Footer.jsx`
3. Edit bio: `src/pages/About.jsx`

### Add Project Screenshots

1. Place images in `public/projects/`
2. Update screenshot names in `src/data/projects.js`
3. Images will display in project cards

### Change Colors

Edit `tailwind.config.js` theme section

### Add New Features

- Create new pages in `src/pages/`
- Create reusable components in `src/components/`
- Add routes in `App.jsx`
- Update navbar with new links
